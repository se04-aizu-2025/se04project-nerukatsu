# Course Structure（講義構成）

## Lectures（講義）

**Human Factors and Managing People in Software Engineering**  
ソフトウェア工学における人間要因と人のマネジメントでは、人間の認知に関する簡単なモデルと、それらがソフトウェアマネージャにどのように関係するかを説明します。

**Software Engineering Management Tools** の講義では、協調的なソフトウェア開発における問題について議論します。

**User Interface Design** および **Visual Programming** に関する講義では、以下を目的とします。

- 一般的な設計原則の提示  
- 人間とコンピュータの相互作用（HCI）スタイルの比較  
- 情報提示スタイルの紹介  
- グラフィカルユーザインタフェース（GUI）の基本的な特徴の説明  

また、**Software Maintenance（ソフトウェア保守）** や  
**Re-engineering（再工学）** を含むソフトウェア進化についても重点的に扱います。

### 講義トピック

- ソフトウェア工学の本質（Nature of the Software Engineering）
- ソフトウェア工学における人間要因と人のマネジメント
- ソフトウェア工学管理ツール
- ソフトウェア検証と妥当性確認（Verification and Validation）
- ソフトウェアテスト
- ユーザインタフェース設計と評価
