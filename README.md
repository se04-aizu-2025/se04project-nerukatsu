# Sorting Visualizer

## Overview
This project is a sorting application with both CUI and GUI.
Users can observe how sorting algorithms work step by step.

## Features
- CUI and GUI support
- Visualized sorting process
- Easy to add new sorting algorithms (extensible design)

## How to Run
### CUI
javac algorithm*.java ui\cui\CliApp.java
java ui.cui.CliApp

### GUI
javac algorithm*.java ui\gui\GuiApp.java
java ui.gui.GuiApp

## Implemented Algorithms
- Bubble Sort
- Selection Sort
- Shaker Sort
